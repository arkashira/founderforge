# 🚀 FounderForge
**Power product managers with prioritized feature roadmaps.** FounderForge is a simple feature roadmap system that allows you to create features with a name, description, effort, and priority, and sorts them by priority and effort.

## Why FounderForge?
* **Prioritized Roadmaps**: Create feature roadmaps sorted by priority and effort to ensure the most important features are tackled first.
* **Effort Estimation**: Estimate the effort required for each feature to help with resource allocation and planning.
* **Feature Description**: Add detailed descriptions to each feature to ensure everyone is on the same page.
* **Customizable**: Customize the feature roadmap to fit your specific needs and workflow.
* **Easy to Use**: Simple and intuitive interface makes it easy to create and manage feature roadmaps.
* **Collaboration**: Invite team members to collaborate on feature roadmaps and ensure everyone is aligned.
* **Data-Driven**: Make data-driven decisions with features sorted by priority and effort.

## Feature Overview
| Feature | Description |
| --- | --- |
| Feature Creation | Create features with name, description, effort, and priority |
| Roadmap Sorting | Sort features by priority and effort |
| Feature Description | Add detailed descriptions to each feature |
| Effort Estimation | Estimate the effort required for each feature |
